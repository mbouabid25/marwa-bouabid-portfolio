Overview

A Chrome extension backed by an AI scoring API that turns live web evidence into a 0–100 ESG risk score for any supplier. Visit a company's website or LinkedIn page, click the extension, and get an automated read on labor, environmental, and governance risk — an early-warning system for procurement teams, without manual research or subjective scoring.

How it works
  • Auto-detection — the extension recognises the company from the page you're on (website, LinkedIn, Wikipedia) and pre-fills the scan.
  • Live web evidence — the API issues focused search queries — labor violations, environmental fines, supply-chain scandals — via Tavily, and dedupes sources.
  • Structured AI extraction — GPT-4o pulls incidents into a strict Pydantic schema with 1–5 severity grades; malformed or invented fields are rejected outright.
  • Deterministic scoring — Python computes the score with recency weighting — an incident this year counts 5× one from three years ago. The LLM never touches the math.

Results & impact

Deterministic scoring produced consistent, repeatable ratings across test suppliers; recency weighting correctly surfaced recent labor disputes over older minor infractions; and schema enforcement eliminated hallucinated fields entirely. For a procurement team: faster vendor diligence, clearer escalation thresholds, and issues surfaced before audits — not after.
