Why this project?

I am from Morocco and we hosted AFCON this year. Attending matches during the winter break was a once-in-a-lifetime experience. I wanted to learn Monte Carlo simulations and live data pipelines, so I built a full tournament prediction engine around it. I ended up learning much more than I planned.

Approach
  • Poisson model — computes time-decayed attack/defence ratings for all 24 teams using Bayesian priors, then uses an 8×8 Poisson goal grid to generate win/draw/loss probabilities, most-likely scorelines, and a 12% host boost for Morocco.
  • CNN model — a 1D convolutional network over team embeddings augmented with form-based numeric features (goals scored/conceded, win rate, deltas). Trained with class-weighted loss and early stopping across 40 epochs.
  • Monte Carlo simulation — 5,000 full tournament runs sampling from Poisson distributions, tracking group finishes, quarter/semi/final appearances, and champion probabilities per team.
  • Live result integration — a dedicated CLI tool ingests real match results from CSV, refreshes group tables, and re-runs all predictions so forecasts stay current as the tournament progresses.
  • Scenario analysis — baseline plus 3 alternative scenarios modelling close-game draws, surfacing how different outcomes reshape the full knockout bracket.
Output

Two Excel workbooks: full match predictions with scenario comparisons and group table projections; plus Monte Carlo stage odds per team. A visualisation module renders champion odds bar charts, per-group finish heatmaps, and competitive-match probability stacked bars.
