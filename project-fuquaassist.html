Overview

FuquaAssist is a Retrieval-Augmented Generation (RAG) chatbot built to answer questions about Duke Fuqua's institutional data — course offerings, policies, faculty, and programs. The challenge: getting a language model to answer accurately about a specific corpus it wasn't trained on, without hallucinating.

How it works
  • Semantic search at scale — thousands of document chunks indexed in a FAISS vector store, searched in under 100ms.
  • Smarter chunking — a bi-encoder architecture (all-mpnet-base-v2) with sliding-window chunking balances context size against retrieval precision.
  • Benchmarked, not assumed — multiple embedding architectures compared against baseline LLMs to pick the best retrieval strategy.
  • Modular pipeline — ingestion → chunking → embedding → indexing → retrieval → generation, each stage swappable.
