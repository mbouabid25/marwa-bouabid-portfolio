Bottom line

Flags high-risk mental-health text in real time — 74% accuracy across seven clinical categories, routing crisis cases to hotline alerts.

Overview

A seven-class early-warning system classifying text into: Suicidal, Depression, Anxiety, Bipolar, Stress, Personality Disorder, and Normal. Trained on anonymised social media posts, forum threads, and diary entries — built for real-time triage that routes the highest-risk cases to immediate intervention.

How it works
  • Order-aware reading — a bi-directional LSTM over a 50-token window captures semantic order (“I am not happy” vs “I am happy”).
  • Built for imbalance — inverse-frequency class-weighted loss handles severe skew in the data (16k Normal samples vs 1.2k Personality Disorder).
  • Clinical granularity — seven classes capture diagnostic nuance beyond simple sentiment, including learned comorbidity patterns.
  • Proactive routing — hotline alerts for high-risk classifications, mindfulness recommendations for lower-risk outputs.

Results & impact

In a live test, “I feel like there is no point in going on…” was correctly ranked Suicidal at 72.8% with Depression (22.8%) as secondary — reflecting learned comorbidity patterns. Paired with routing, that makes the difference between a dashboard metric and a hotline call that happens in time.
